Data files for the gap-size sweep go here.

Expected CSV format (two columns):
  frequency_THz,S21_dB_norm

Example filenames:
  gap_5um_norm.csv
  gap_3um_norm.csv
  gap_1um_norm.csv
  gap_0p8um_norm.csv

Export from Lumerical using the S21 frequency monitor results,
then normalise against the reference straight-CPW simulation.
